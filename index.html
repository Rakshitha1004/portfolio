Projects

UPI ID Fraud Detection Using Machine Learning

This project focuses on detecting fraudulent transactions in UPI payment systems using machine learning techniques. The system analyzes transaction data such as transaction amount, frequency, location patterns, and user behavior to identify suspicious activities. Various machine learning algorithms are used to train the model on historical transaction data so that it can accurately classify transactions as legitimate or fraudulent. The goal of this project is to improve digital payment security by reducing financial fraud and providing early detection of suspicious transactions in real time.

Online Unused Medicine Donation System

The Online Unused Medicine Donation System is a web-based platform designed to reduce medicine wastage and help people in need. The system allows users to donate unused and unexpired medicines through an online portal. Donors can upload medicine details such as name, expiry date, and quantity, while registered NGOs or healthcare organizations can request and collect these medicines for redistribution to needy patients. This platform helps promote responsible medicine disposal, supports healthcare accessibility for underprivileged communities, and encourages community-driven healthcare support.
